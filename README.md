## Project Description

This project analyzes Python as an open-source software. It includes its origin, licensing, ethical considerations, Linux footprint, and comparison with proprietary software.

## Scripts Included

### Script 1 — System Identity Report

Displays system details like user name and date.

### Script 2 — Package Inspector

Checks whether Python is installed and shows its version.

### Script 3 — Disk and Permission Auditor

Displays directory sizes and permissions.

### Script 4 — Log File Analyzer

Counts occurrences of a keyword in a log file.

### Script 5 — Manifesto Generator

Generates a personalized open-source manifesto using user input.
